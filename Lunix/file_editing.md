🟢 Nano Quick Start
        nano file.txt

⌨️ Basic Controls (remember this)
        Ctrl = ^ (shown in nano)
        Example: ^X = Ctrl + X

✍️ Editing
        Just start typing (no modes like Vim)
        Arrow keys → move cursor
        Backspace → delete
        Enter → new line

💾 File Operations
        Ctrl + O → Save (Write Out)
        Enter → confirm filename
        Ctrl + X → Exit
        Ctrl + X → then Y → save & exit


🔍 Search & Replace
        Ctrl + W → Search
        Type word → Enter
        Alt + W → next match
        Ctrl + \ → Replace
        Enter old word → new word


✂️ Copy / Paste
        Ctrl + ^ → Start selection
        Move cursor → select text
        Alt + 6 → Copy
        Ctrl + K → Cut
        Ctrl + U → Paste

        
🔄 Undo / Redo
        Alt + U → Undo
        Alt + E → Redo

📄 Navigation
        Ctrl + A → Start of line
        Ctrl + E → End of line
        Ctrl + Y → Page up
        Ctrl + V → Page down

❌ Cancel / Help
        Ctrl + C → Show cursor position
        Ctrl + G → Help
        Ctrl + X → Exit

🧠 Golden Rules
        No modes → just type
        Commands always shown at bottom
        If stuck → Ctrl + X to exit

        
🚀 Ultra-Short Memory Version
        Open → nano file.txt
        Save → Ctrl + O
        Exit → Ctrl + X
        Search → Ctrl + W
        Cut → Ctrl + K
        Paste → Ctrl + U
